------------------------------------------------------------------------------------------------------
- Set up localhost server to run this app.
- Takes input name of a song , searches it from an online database and downloads  a suitable video.
- Strips the audio and mutes the vocals , then asks the user to sing .
- FFT analysis (Energy Density Waveforms) used to determine the score .

-Python,Perl and Bash scripts for downloading and manipulating files . 
- sox and aubiopitch for audio manipulation
- PHP for server side scripting .
